# Node.js API 

An API with JWT to which can be used as a plugin for other projects.

## Tech Used

- Node.js
- ExpressJS

## External Libraries
- JOI (For Validation)
- JWT (JSON Web Tokens)

## Installation

```bash
npm install

// Now start the server
npm start
```

## Usage

- It can be self hosted.
- It can be easily modify it as per your use and use it.

## Testing Tools

- Postman
